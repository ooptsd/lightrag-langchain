# Requirements Archive: v1.0

## v1 Requirements — All Complete

### STOR — 数据层

- [x] **STOR-01**: 从 PGVector 读取 entities_vdb（entity_name, content, source_id, file_path）
- [x] **STOR-02**: 从 PGVector 读取 relationships_vdb（src_id, tgt_id, content, keywords, weight）
- [x] **STOR-03**: 从 PGVector 读取 chunks_vdb（content, full_doc_id, chunk_order_index, file_path）
- [x] **STOR-04**: 从 Apache AGE 图数据库读取 entity 节点和 relation 边

### QUERY — 查询模式

- [x] **QUERY-01**: Naive 模式 — 纯向量相似度搜索 chunks_vdb
- [x] **QUERY-02**: Local 模式 — entities_vdb 向量搜索 → AGE 图扩展
- [x] **QUERY-03**: Global 模式 — relationships_vdb 向量搜索 → AGE 图查找
- [x] **QUERY-04**: Hybrid 模式 — local + global 并行，round-robin 合并
- [x] **QUERY-05**: Mix 模式 — hybrid + chunks_vdb 向量搜索
- [x] **QUERY-06**: Bypass 模式 — 无检索，LLM 直接回答

### RETR — Retriever 接口

- [x] **RETR-01**: 6 种 BaseRetriever 子类（Naive/Local/Global/Hybrid/Mix/Bypass）
- [x] **RETR-02**: sync invoke() + async ainvoke() + List[Document] 返回
- [x] **RETR-03**: metadata 含 source_id, file_path, entity/relation 引用

### CHAIN — Chain 接口

- [x] **CHAIN-01**: 完整 LCEL Chain — 查询 → 关键词 → 检索 → 上下文 → LLM 生成
- [x] **CHAIN-02**: invoke / ainvoke / astream 三种调用方式
- [x] **CHAIN-03**: QueryParam 支持 hl_keywords / ll_keywords 跳过关键词提取

### CONF — 配置

- [x] **CONF-01**: .env PostgreSQL 连接配置
- [x] **CONF-02**: .env LLM 配置
- [x] **CONF-03**: .env Embedding 配置
- [x] **CONF-04**: .env Reranker 配置
- [x] **CONF-05**: .env 查询参数配置

### LLM — LLM / Embedding / Reranker 集成

- [x] **LLM-01**: ChatOpenAI 兼容接口 — 所有 OpenAI API 格式 provider
- [x] **LLM-02**: OpenAIEmbeddings 兼容接口
- [x] **LLM-03**: 多 Reranker 支持 — aliyun / cohere / jina
- [x] **LLM-04**: LLM 关键词提取 — high-level + low-level keywords
- [x] **LLM-05**: Token 预算控制 — 动态分配 entity/relation/chunk 配额
