       - `base_url=self._config.binding_host`

       - `check_embedding_ctx_length=False` — required for non-OpenAI providers (per RESEARCH.md Code Examples)

    - Use `from langchain_openai import ChatOpenAI` (not `langchain.chat_models`) — per RESEARCH.md Standard Stack
    - `get_secret_value()` is called on `SecretStr` to extract the raw string for the constructor
