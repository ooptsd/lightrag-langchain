    `_aget_relevant_documents(query: str, *, run_manager=None) -> list[Document]`:

    3. `result: QueryResult = await naive_strategy(embedding, vector_store=self.vector_store, chunk_top_k=self.chunk_top_k)`
    4. Convert chunks: `[chunk_to_document(c, retrieval_mode="naive") for c in result.chunks]`
    5. Return chunk documents list (entities/relations/triples are empty for naive)

    **Class 2: LocalRetriever(LightRAGBaseRetriever)**

    RETR-01 (local mode). Requires both vector_store and graph_store.

    `_aget_relevant_documents(query: str, *, run_manager=None) -> list[Document]`:
    1. `embedding = self._generate_query_embedding(query)` → list[float]
    2. Import and call `from lightrag_langchain.query.strategies import local_strategy`
    3. `result = await local_strategy(embedding, vector_store=self.vector_store, graph_store=self.graph_store, top_k=self.top_k)`
    4. Build lookups: `node_lookup, edge_lookup = build_graph_lookups(result.graph_triples)`
    5. Entity Documents: iterate result.entities in order
       - Match key: `entity.entity_name` in `node_lookup` (entity_name from VDB == entity_id from AGE per RESEARCH.md Pitfall 1)
       - If match found: `node = node_lookup[entity.entity_name]; doc = entity_to_document(entity, entity_type=node.entity_type, description=node.description, retrieval_mode="local")`
       - If no match: `doc = entity_to_document(entity, retrieval_mode="local")` (entity_type/description default to "")
    6. Triple Documents: `[graph_triple_to_document(t, retrieval_mode="local") for t in result.graph_triples]`
    7. Return: entity_docs + triple_docs (concatenated, entities first per upstream order)

    **Class 3: GlobalRetriever(LightRAGBaseRetriever)**

    RETR-01 (global mode). Requires both vector_store and graph_store.

    `_aget_relevant_documents(query: str, *, run_manager=None) -> list[Document]`:
    1. `embedding = self._generate_query_embedding(query)` → list[float]
    2. Import and call `from lightrag_langchain.query.strategies import global_strategy`
    3. `result = await global_strategy(embedding, vector_store=self.vector_store, graph_store=self.graph_store, top_k=self.top_k)`
    4. Build lookups: `node_lookup, edge_lookup = build_graph_lookups(result.graph_triples)` — only edge_lookup is used; node_lookup is available but not needed for relation enrichment
    5. Relation Documents: iterate result.relations in order
       - Match key: `(relation.src_id, relation.tgt_id)` in `edge_lookup`
       - If match found: `edge = edge_lookup[(relation.src_id, relation.tgt_id)]; doc = relation_to_document(relation, keywords=edge.keywords or "", weight=edge.weight, source_id=edge.source_id or "", retrieval_mode="global")`
       - If no match: `doc = relation_to_document(relation, retrieval_mode="global")` (uses relation's own keywords/weight which may be None from VDB)
    6. Triple Documents: `[graph_triple_to_document(t, retrieval_mode="global") for t in result.graph_triples]`
    7. Return: relation_docs + triple_docs (concatenated, relations first per upstream order)
